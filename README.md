# CodeAlpha Flashcard Quiz App 📚

A professional, interactive Flashcard Quiz application built with Flutter. This project is part of my **App Development Internship at CodeAlpha**.

## 🚀 Features

* **Dynamic Flashcards:** Create, edit, and delete flashcards easily.
* **Flip Animation:** Modern 3D flip effect to reveal answers.
* **Smooth Navigation:** Uses `PageView` for a seamless experience between cards.
* **Progress Tracking:** Visual progress bar to track your learning session.
* **Modern UI:** Clean, professional, and user-friendly interface.

## 🛠 Tech Stack

* **Framework:** Flutter
* **Language:** Dart
* **State Management:** Provider
* **Architecture:** Clean Architecture (Separation of Concerns)

* `lib/models`: Data structures.
* `lib/providers`: Logic and state management.
* `lib/screens`: Main UI screens.
* `lib/widgets`: Reusable UI components.
